University Student Entrepreneurship Data Analysis

 Project Overview
This project focuses on analyzing university student entrepreneurship data using Python libraries.

Technologies Used
. Python
. Pandas
. NumPy
. Matplotlib
. Seaborn

Exploratory Data Analysis 
The project includes:
. Data Cleaning
. Data Visualization
. Statistical Analysis

 Visualizations
The following plots were created:
. Count Plot
. Bar Plot
. Histogram
. KDE Plot
. Box Plot
. Scatter Plot
. Heatmap
. Pair Plot

 Conclusion
This analysis helps understand relationships between student characteristics such as Age, Perseverance, Competitiveness, and SelfConfidence.
